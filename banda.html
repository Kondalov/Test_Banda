git init - инициализирует папку или создаёт новый файл

git add & git . - добавляет(записывает) контрольную точку указанного файла Например: git add index.html

git status - позволяет узнавать статус контрольной точки. Если были внесены изменения, то git status подсветит файл красным цветом.
