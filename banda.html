git init - инициализирует папку или создаёт новый файл

git add & git . - добавляет(записывает) контрольную точку указанного файла Например: git add index.html

git status - позволяет узнавать статус контрольной точки. Если были внесены изменения, то git status подсветит файл красным цветом.

git commit -m "your text" - записываем любой комминтарий по изменениям в коде

git branch - позволяем узнать статус ветки Чтобы создать новую ветку, нужно прописать: git branch name git branch -D имя ветки - удаляет ветку git checkout Nikolay - таким образом можно переключатся между ветками git checkout -b имя ветки - создаём и сразу переходим на новую ветку

git merge имя ветки - объеденяет ветки контрольными точками
